# cnsite-aiyouxi-resource

## 项目简介

`cnsite-aiyouxi-resource` 是一个用于归档和发布多个独立 HTML 页面的仓库。本仓库不针对任何特定域名或具体网站，仅作为静态资源的集合存储与分发用途。所有页面均为独立文件，便于直接访问或嵌入其他项目使用。

## 目录说明

仓库根目录下包含以下内容：

- `index.html`：项目入口页面（可选）
- `pages/`：存放各独立 HTML 页面的目录
  - 每个页面以单独文件形式保存，命名格式为 `page-xxx.html`
- `assets/`：存放页面所需的静态资源（如 CSS、JS、图片等）
- `README.md`：本文件，项目说明文档

目录结构示例：

```
cnsite-aiyouxi-resource/
├── README.md
├── index.html
├── pages/
│   ├── page-001.html
│   └── page-002.html
└── assets/
    ├── style.css
    └── script.js
```

## 页面归档说明

本仓库中的 HTML 页面均为独立归档文件，不依赖于外部域名或特定服务器环境。每个页面及其相关资源均打包在仓库内，以保证在任意环境下均可直接运行。

归档页面内容可能涉及多种类型，包括但不限于信息展示、工具页面、演示页面等。所有页面均以静态 HTML 形式存在，不包含动态后端逻辑。

## 使用方式

- 直接访问 `pages/` 目录下的 HTML 文件即可查看对应页面内容。
- 如需部署，可将整个仓库克隆或下载到本地，或通过 GitHub Pages 等静态托管服务发布。
- 页面中引用的资源文件请保持相对路径不变，以确保正常加载。
